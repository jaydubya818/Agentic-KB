# 19 Open-Source GitHub Repos for AI Agents

## Source

Curated LinkedIn post captured from Apple Notes on April 5, 2026.

## Multi-Agent Orchestration

- **Camel-AI** — https://github.com/camel-ai/camel
  Role-playing agents with structured conversations and task delegation.

- **MetaGPT** — https://github.com/geekan/MetaGPT
  Multi-agent meta-programming framework; assigns specialized roles (PM, Engineer, and/or QA) to LLMs. 

- **Atomic Agents** — Modular agent components for composing complex agent pipelines from small, testable units.

- **GPTSwarm** — Graph-based multi-agent framework for optimizable agent networks.

- **Burr** — State-machine-based agent framework for building reliable, inspectable agentic workflows.

## Autonomous Builders (Coding Agents)

- **SWE-agent (Princeton)** — Autonomous software engineering agent for solving GitHub issues.

- **OpenHands** — Open-source Devin alternative; autonomous coding and task execution.

- **Agent-Zero** — Minimal general-purpose autonomous agent.

- **Devika** — AI software engineer that plans, codes, and executes.

- **Plandex** — Terminal-based AI coding agent for large, multi-file tasks.

## Memory & Reasoning

- **DSPy (Stanford)** — Framework for programming (not prompting) language models; self-optimizing pipelines.

- **mem0** — https://github.com/mem0ai/mem0
  Persistent memory layer for AI agents; stores and retrieves cross-session context.

- **MemGPT / Letta** — https://github.com/cpacker/MemGPT
  OS-inspired memory management for LLMs; hierarchical memory with self-managed paging.

- **Storm (Stanford)** — Research synthesis agent for generating Wikipedia-style long-form articles.

## Tools & Observability

- **Skyvern** — Browser automation for agents using LLMs + computer vision (replaces Playwright/Selenium scripts).

- **E2B** — https://github.com/e2b-dev/E2B
  Code sandboxes for AI agents; secure execution environments for agent-generated code.

- **AgentOps** — Agent observability and monitoring platform; session replays, cost tracking, error debugging.

- **Composio** — Integration platform for agents; 100+ pre-built tool integrations (GitHub, Notion, Salesforce, etc.)

## Notes

- All listed repos are free/open-source. Total cost: $0.
- Represents a comprehensive coverage of the agentic stack: orchestration, autonomous execution, memory management, and production tooling.
- Sources for expanded context: Camel-AI and MetaGPT are the most established; DSPy is academically rigorous; E2B and AgentOps are production-ready infrastructure.
