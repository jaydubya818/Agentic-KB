# Source Evaluation

How to judge if a source is worth trusting. Apply to EVERY source before using it in analysis.

*Maps to Agentic-KB `system/policies/source-trust-policy.md` — Tier 1 here ≈ "canonical" there.*

---

## Source Tier System

### Tier 1: Primary Data (highest trust)
- Raw datasets (UN, World Bank, national statistics offices, CDC, IMF)
- Peer-reviewed studies with full methodology visible
- Financial filings, government records, court documents
- Direct measurements and observations
- **USE FOR:** [[knowledge/data-points]], hard claims, base assumptions
- **Trust in synthesis:** state directly, no hedging required

### Tier 2: Expert Analysis
- Reports from domain-specific research institutions (Brookings, RAND, McKinsey Global Institute)
- Books by recognized authorities in the field
- Long-form investigative journalism with cited sources (NYT, FT, The Atlantic deep dives)
- Conference papers and working papers with methodology
- **USE FOR:** interpretation, causal claims, framework building
- **Trust in synthesis:** "Research from [institution] finds..." with caveat if single source

### Tier 3: Informed Commentary
- Expert blog posts and newsletters (Stratechery, Substack from named domain experts)
- Quality podcasts with domain experts (Lex Fridman, Dwarkesh Patel, EconTalk)
- Think tank reports — CHECK FUNDING SOURCES before using
- Industry publications (trade press, analyst reports)
- **USE FOR:** angles you hadn't considered, hypothesis generation only
- **Trust in synthesis:** "One analyst argues..." — never as evidence, only as framing

### Tier 4: General Media
- Major news outlets (breaking coverage only, not analysis)
- Wikipedia (good for orientation, never for final claims)
- Popular science writing
- **USE FOR:** initial orientation only. always verify upstream to Tier 1 or 2

### Tier 5: Social/Anecdotal (lowest trust)
- Twitter/X threads, Reddit posts
- Personal anecdotes, testimonials
- Viral content, hot takes
- **USE FOR:** signal detection only — "people are talking about X" ≠ "X is true"

---

## Red Flags (auto-downgrade any source by 1 tier if present)
- No cited sources or methodology visible
- Author has financial incentive in the conclusion
- Published by organization with known agenda on the topic
- Cherry-picked time frames or geographies
- Conflates correlation with causation
- Uses emotional language instead of evidence
- Can't find the original data — only summaries of summaries

---

## Evaluation Checklist
For every key claim before using it in analysis:
1. What tier is this source?
2. Can I find the same claim in a Tier 1 or 2 source?
3. Who funded this research or publication?
4. What would the author lose if they were wrong?
5. Is this the BEST available evidence, or just the first I found?

See [[methodology/contradiction-protocol]] when two credible sources disagree.
Record all Tier 1/2 verified claims in [[knowledge/data-points]].
